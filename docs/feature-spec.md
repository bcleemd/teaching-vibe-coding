# Feature Specification: 코딩 초보자를 위한 바이브코딩 교육 자료

## 1. 개요 (Overview)
- **목적**: 코딩 경험이 없거나 거의 없는 초보자를 대상으로 2시간 이내에 CLI 설치부터 AI 협업 개발(바이브코딩)의 개념을 학습할 수 있는 가이드라인과 상세 강의안을 제공한다.
- **대상**: 코딩 입문자, 터미널 환경이 낯선 학습자
- **범위**: 1부(CLI 설치 및 환경 구성) 및 2부(바이브코딩 개념 및 첫 AI 협업) 상세 강의안 작성

## 2. 문서 구조 및 아키텍처
본 프로젝트는 다음과 같은 마크다운 파일 구조로 구성된다.
1. `README.md`: 교육 개요, 타임라인, 대상자 준비물 요약
2. `session1_cli_setup.md`: 1부 상세 강의안 (터미널 기초, CLI 설치(Node.js, Git) 및 확인)
3. `session2_vibe_coding_intro.md`: 2부 상세 강의안 (바이브코딩 개념, 프롬프트 작성 팁, 첫 AI 결과물 확인 및 실행 방법)

## 3. 기능 및 콘텐츠 요구사항 (Feature Requirements)

### 3.1. README.md
- 전체 2시간 교육의 타임라인 및 교육 철학 명시.
- 준비 사항 가이드.

### 3.2. Session 1: CLI 설치 및 기초
- OS별(macOS, Windows) 터미널 접속 방법.
- Node.js & npm 설치 방법 (초보자용 공식 웹사이트 다운로드 방식).
- Git 설치 방법.
- 환경 검증을 위한 필수 명령어 실습 (`node -v`, `npm -v`, `git --version`, `pwd` / `cd`, `ls` / `dir`).

### 3.3. Session 2: 바이브코딩 입문
- 바이브코딩(Vibe Coding)의 정의 및 필요성.
- AI 코딩 어시스턴트와의 대화법 (역할 부여, 명확한 지시, 제약 조건).
- 첫 협업 실습: "자기소개 웹페이지(HTML)" 요청 프롬프트 템플릿 제공.
- AI 코드를 내 컴퓨터에 저장하고 웹 브라우저로 열어 실행하는 구체적인 실습 단계 가이드.
