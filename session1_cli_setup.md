# 1부: CLI 설치 및 기초 환경 설정 가이드

개발을 시작할 때 초보자들이 가장 무서워하는 것이 검은 화면, 즉 **터미널(Terminal)**입니다. 이 세션에서는 검은 화면의 공포를 극복하고, AI 코딩 도구들이 정상 작동할 수 있도록 컴퓨터의 기초 체력을 다지는 CLI(Command Line Interface) 설치 과정을 진행합니다.

---

## 1. 터미널(Terminal)이란 무엇인가요?

우리는 보통 컴퓨터를 쓸 때 마우스로 아이콘을 더블클릭하고 폴더를 엽니다. 이를 **GUI(Graphic User Interface)**라고 합니다.
반면, 마우스 없이 오직 **글자(명령어)를 키보드로 타이핑해서 컴퓨터를 조종하는 환경**을 **CLI(Command Line Interface)**라고 합니다.

### 내 컴퓨터에서 터미널 켜기
- **Windows (윈도우)**: 
  윈도우는 크게 두 가지 기본 터미널을 제공합니다. 현대적인 개발 환경을 위해 더 강력한 **PowerShell(파워쉘)** 사용을 권장합니다.

  - **방법 A: PowerShell (파워쉘 - 권장)**
    1. 키보드의 `시작(Win)` 키를 누르고 `PowerShell` 또는 `파워쉘`을 검색합니다.
    2. 마우스 우클릭 후 **'관리자 권한으로 실행'**을 선택해 켭니다.

    > 💡 **PowerShell 최신 버전(PowerShell 7)으로 업그레이드하기**
    > 윈도우에 기본 내장된 파워쉘(버전 5.1)은 구형이라 최신 CLI 도구와 호환성 문제가 생길 수 있습니다. 아래 명령어로 즉시 최신 버전으로 업그레이드할 수 있습니다.
    > 1. 파워쉘 창에 다음 명령어를 입력하고 엔터를 칩니다:
    >    ```powershell
    >    winget install --id Microsoft.Powershell --source winget
    >    ```
    > 2. 설치가 완료된 후 터미널 창을 완전히 닫고 다시 열면, `pwsh` 명령어로 최신 파워쉘을 실행할 수 있습니다.

  - **방법 B: cmd (명령 프롬프트)**
    1. 키보드의 `시작(Win)` 키를 누르고 `cmd` 또는 `명령 프롬프트`를 검색합니다.
    2. 마우스 우클릭 후 **'관리자 권한으로 실행'**을 선택해 켭니다.

- **macOS (맥)**:
  1. `Command(⌘) + Space`를 눌러 스포트라이트(Spotlight)를 켭니다.
  2. `Terminal` 또는 `터미널`을 입력하고 엔터를 칩니다.

## 2. 필수 CLI 도구 설치하기

바이브코딩 및 향후 현대적인 개발 환경을 위해 아래 두 가지 도구를 설치해야 합니다.

### ① Node.js & npm 설치 (자바스크립트 실행 환경)
Node.js는 우리가 작성할 코드가 컴퓨터에서 동작할 수 있게 해주는 엔진입니다. npm은 다른 사람들이 미리 만들어 둔 유용한 코드 뭉치(라이브러리)를 쉽게 가져오게 해주는 도구로, Node.js를 설치하면 자동으로 같이 설치됩니다.

1. **다운로드**: [Node.js 공식 홈페이지 (LTS 버전)](https://nodejs.org/)에 접속합니다.
2. **설치**: 화면에 나오는 초록색 버튼 중 **LTS(보장된 안정 버전)** 버튼을 눌러 설치 파일을 다운로드하고 실행합니다.
3. **설치 진행**: 설치 마법사 창에서 모두 `Next(다음)` 또는 `Install(설치)`을 눌러 기본값으로 완료합니다.

### ② Git 설치 (버전 관리 도구)
개발자들의 코드 타임머신이자 협업 도구인 Git을 설치합니다.

- **Windows**:
  1. [Git for Windows](https://git-scm.com/download/win)에 접속하여 64-bit Git for Windows Setup 파일을 다운로드합니다.
  2. 설치 옵션이 많지만, 초보자의 경우 모두 기본값(Next)으로 두고 끝까지 설치를 완료합니다.
- **macOS**:
  1. 터미널을 열고 `git --version`을 입력합니다.
  2. 만약 설치되어 있지 않다면 자동으로 Xcode 명령줄 도구(Command Line Tools)를 설치하겠냐는 팝업이 뜹니다. **'설치'** 버튼을 누릅니다.

### ③ uv 설치 및 이를 이용한 Python 설치 (파이썬 개발 환경)
`uv`는 파이썬(Python) 패키지 설치, 가상환경 관리, 그리고 파이썬 자체의 설치와 버전 관리를 믿을 수 없을 정도로 빠르고 쉽게 해결해 주는 현대적인 파이썬 올인원(All-in-one) 도구입니다. 복잡한 파이썬 환경 설정과 버전 충돌 문제를 깔끔하게 해결해 줍니다.

- **uv 설치하기**:
  터미널을 열고 컴퓨터 환경에 맞는 명령어를 입력한 뒤 엔터를 칩니다.
  - **macOS / Linux**:
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
  - **Windows**:
    PowerShell(파워쉘)을 **관리자 권한으로** 열고 아래 명령어를 입력합니다.
    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```
    *(참고: 환경에 따라 `winget install --id Astral.uv` 명령어를 대신 사용할 수도 있습니다.)*

- **uv로 Python 설치하기**:
  `uv` 설치가 완료되었다면, 터미널 명령어 한 줄로 원하는 버전의 파이썬을 즉시 설치할 수 있습니다. 시스템에 지저분하게 파이썬을 깔지 않고, 안전하게 격리된 최신 파이썬 환경을 구축할 수 있습니다.
  - **최신 파이썬 설치**:
    ```bash
    uv python install
    ```
  - **특정 파이썬 버전 설치 (예: 3.12)**:
    ```bash
    uv python install 3.12
    ```

> 💡 **중요**: 모든 설치가 완료되면, **열려 있던 터미널 창을 완전히 닫고 새로 열어야** 방금 설치한 도구들을 컴퓨터가 인식합니다.

## 3. 설치 확인 및 터미널 명령어 실습

새로 열린 터미널 창에 아래 명령어를 하나씩 타이핑하고 엔터를 쳐보세요. 버전 숫자가 정상적으로 출력되면 설치 성공입니다.

```bash
# Node.js 버전 확인
node -v

# npm 버전 확인
npm -v

# Git 버전 확인
git --version

# uv 버전 확인
uv --version

# uv로 설치한 파이썬 버전 확인
uv run python --version
```

### 🏃‍♂️ 왕초보 필수 터미널 명령어 3가지
폴더를 탐색하는 마우스 동작을 터미널 명령어로 직접 따라 해봅시다.

1. **현재 내가 있는 폴더 위치 확인하기**
   - **macOS**: `pwd` (Print Working Directory)
   - **Windows**: `cd` (인자 없이 입력하면 현재 경로 출력)
2. **현재 폴더 안에 있는 파일/폴더 목록 보기**
   - **macOS**: `ls` (List)
   - **Windows**: `dir` (Directory)
3. **다른 폴더로 이동하기**
   - 명령어: `cd 폴더이름` (Change Directory)
   - 상위 폴더로 가려면: `cd ..`
   - 바탕화면으로 가려면: `cd Desktop` (mac) 또는 `cd OneDrive\Desktop` / `cd Desktop` (win)

## 🚨 자주 발생하는 트러블슈팅 (에러 극복하기)

- **`'node'` 또는 `'uv'`가 내부 또는 외부 명령, 실행할 수 있는 프로그램이 아니라는 에러**
  - 원인: 도구가 설치되지 않았거나, 설치 후 터미널을 새로고침(재실행)하지 않았을 때 발생합니다.
  - 해결법: 터미널 창을 완전히 종료한 후 다시 켜고 명령어를 입력해 보세요. 특히 `uv` 설치 후에는 터미널 재실행이 필수적이며, 윈도우 환경인 경우 환경 변수(`PATH`)가 아직 완전히 갱신되지 않았을 수 있으니 터미널 재실행 후에도 안 되면 시스템을 다시 시작해 보세요.
