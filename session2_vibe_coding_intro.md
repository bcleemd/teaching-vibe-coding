# 2부: 바이브코딩 입문 및 첫 AI 협업 가이드

이 세션에서는 코드를 타이핑하기 전, AI 어시스턴트를 마치 내 비서이자 유능한 개발 파트너처럼 다루는 **바이브코딩(Vibe Coding)**의 핵심 개념을 배우고 직접 첫 결과물을 만들어봅니다.

---

## 1. 바이브코딩(Vibe Coding)이 무엇인가요?

과거의 개발이 문법책을 외우고 오타 하나 때문에 밤을 새우는 고단한 작업이었다면, **바이브코딩**은 전체적인 흐름과 설계(Vibe)를 개발자가 쥐고, 실제 손가락 노동(코드 작성)은 AI에게 맡기는 새로운 개발 패러다임입니다.

### 초보자에게 바이브코딩이 강력한 이유
- **문법 몰라도 시작 가능**: 괄호나 세미콜론이 어디 들어가는지 몰라도 됩니다.
- **빠른 성취감**: 20분 만에 화면에 나타나는 나만의 웹사이트를 보고 코딩에 흥미를 붙일 수 있습니다.
- **배우면서 만들기**: AI가 짠 코드를 뜯어보며 "아, 이게 이렇게 동작하는구나"를 실전에서 배웁니다.

---

## 2. AI를 조종하는 프롬프트 작성 3대 법칙

AI에게 단순히 *"웹사이트 하나 만들어줘"* 라고 하면 엉망진창인 결과가 나옵니다. 유능한 선배 개발자처럼 AI를 지휘하려면 아래 3대 법칙을 지켜야 합니다.

### 💡 법칙 1: 역할 부여하기 (Role Playing)
AI에게 지금 어떤 역할을 해야 하는지 지정하세요.
> **예시**: "너는 웹 프론트엔드 전문 개발자야. 코딩 초보자인 나를 위해 친절하고 가독성이 뛰어난 코드를 짜줘."

### 💡 법칙 2: 구체적인 제약 조건 제공 (Constraint)
한 화면에 다 들어갈 수 있게 가볍고 심플한 환경으로 요청합니다.
> **예시**: "HTML, CSS, JS 코드를 별도의 파일로 나누지 말고, **오직 하나의 `index.html` 파일** 안에 모두 담아줘. 외부 라이브러리는 쓰지 마."

### 💡 법칙 3: 점진적으로 만들기 (Step-by-Step)
한 번에 10가지 기능을 요구하면 AI도 헷갈려합니다. 먼저 핵심 기능 1개를 만들고, 그 위에 살을 붙여갑니다.
> **예시 (1단계)**: "버튼을 누르면 화면에 오늘 어울리는 패션 아이템이 나오는 화면을 만들어줘."
> **예시 (2단계)**: "방금 만든 화면에 모던한 네이비 톤의 다크모드 디자인(CSS)을 추가해줘."

---

## 3. [실습] 15분 만에 자기소개 페이지 만들기

실제로 AI(ChatGPT, Claude, Gemini 등)에게 다음 프롬프트를 복사해서 붙여넣고 질문해 봅시다.

### 📝 복사할 프롬프트 예시
```text
너는 초보 개발자를 돕는 친절한 시니어 웹 개발자야. 
나를 소개하는 아름다운 웹페이지를 만들고 싶어. 

아래 요구사항에 맞추어 하나의 HTML 파일로 코드를 작성해줘.

[요구사항]
1. 화면 중앙에 내 프로필(이름: 홍길동, 직업: 예비 바이브코더, 취미: 독서)이 카드 형태로 깔끔하게 보이게 해줘.
2. 아래에는 "나의 다짐 보기"라는 버튼이 있게 해줘.
3. 이 버튼을 클릭하면, 미리 설정된 멋진 동기부여 문구 3개 중 하나가 화면에 랜덤으로 나타나도록 해줘.
4. CSS 스타일은 아주 모던하고 프리미엄한 느낌(그레이와 파스텔 블루 톤, 둥근 모서리, 부드러운 그림자 효과)으로 작성해줘.
5. HTML 문서 하나(`index.html`)에 CSS(<style>)와 JS(<script>)를 모두 포함시켜서 한 번에 실행할 수 있게 해줘.
```

---

## 4. AI가 준 코드를 실행하는 법 (VS Code 실습)

AI가 코드를 짜주었다면, 이제 내 컴퓨터에서 실행해 볼 시간입니다.

1. **폴더 만들기**: 바탕화면이나 적당한 위치에 `my-first-vibe`라는 이름의 빈 폴더를 만듭니다.
2. **VS Code 실행**: VS Code를 열고, `파일(File) -> 폴더 열기(Open Folder)`를 눌러 방금 만든 `my-first-vibe` 폴더를 선택합니다.
3. **새 파일 생성**: 왼쪽 탐색기 빈 곳을 마우스 우클릭하거나 새 파일 아이콘을 눌러 **`index.html`** 이라는 이름의 파일을 만듭니다.
4. **코드 붙여넣기**: AI가 작성해 준 코드 블록 우측 상단의 `Copy code`를 눌러 복사한 뒤, VS Code의 `index.html` 파일 안에 붙여넣고 `Ctrl + S` (mac은 `Cmd + S`)로 저장합니다.
5. **브라우저로 실행하기**:
   - 파일 탐색기에서 `index.html` 파일을 더블클릭합니다.
   - 크롬이나 사파리 같은 웹브라우저 창이 열리면서 내가 요청한 페이지가 마법처럼 켜집니다! 버튼도 클릭해 보세요.

## 🚀 한 걸음 더: 마음에 들지 않는 부분 고치기 (티키타카)

브라우저에서 확인한 결과물이 2% 아쉽다면 AI에게 추가 피드백을 줍니다.

- **디자인 수정 요청**: *"전체 배경색을 좀 더 짙은 네이비색 다크모드로 바꿔주고, 글씨 색상은 흰색으로 수정해줘."*
- **기능 추가 요청**: *"카드를 클릭하면 부드럽게 360도 회전하는 애니메이션 효과를 추가해줘."*

이처럼 AI에게 계속 말을 걸며 내가 원하는 완성도로 빚어가는 과정이 바로 **바이브코딩의 진정한 묘미**입니다.
