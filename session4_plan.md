# Session 4 실습 계획서: 유튜브 1000문장 영어 학습 Q&A 앱

본 계획서는 유튜브 인기 영어 학습 영상인 [[생활영어 1000문장 그냥 틀어두십쇼 1,2,3편 몰아듣기](https://www.youtube.com/watch?v=_50HS70urTw)]에 수록된 문장을 기반으로, 브라우저 환경에서 작동하는 프리미엄 영어 회화 학습 대시보드 애플리케이션을 구축하기 위한 설계도입니다.

---

## 1. 프로젝트 개요

*   **프로젝트 명**: Vibe English 1000 (영어회화 1000문장 학습기)
*   **주요 목적**: 
    *   유튜브 영상([생활영어 1000문장 그냥 틀어두십쇼](https://www.youtube.com/watch?v=_50HS70urTw))의 1000문장 데이터를 로컬 JSON 데이터베이스로 구축.
    *   CLI 실행 환경(로컬 웹 서버)을 학습 목적에 연동하여 CORS 이슈를 극복하고 데이터를 비동기 로딩.
    *   플래시카드(암기), 타이핑 퀴즈(학습), 음성 듣기(TTS) 기능을 갖춘 미려한 웹 애플리케이션 개발.
*   **기술 스택**: HTML5, Vanilla CSS3 (글래스모피즘 & 모던 다크 테마), Vanilla JavaScript (ES6+), JSON 데이터베이스.

---

## 2. 파일 구조 (Architecture)

프로젝트 루트 내 `session4_english_app` 폴더를 생성하고 아래 구조로 구성합니다.

```text
session4_english_app/
├── index.html          # 메인 UI 구조 및 마크업
├── style.css           # 모던 디자인 시스템 (다크/라이트 테마, 애니메이션)
├── app.js              # 비동기 데이터 fetch, 상태 관리, 퀴즈 및 오디오 로직
└── sentences.json      # 1000문장 Q&A 데이터베이스 (JSON 포맷)
```

---

## 3. 핵심 기능 정의

### ① JSON 데이터베이스 연동
*   `sentences.json`에 문장 번호, 영어 문장, 한글 해석, 상황별 팁/힌트 데이터를 구조화하여 저장.
*   `app.js`에서 `fetch()` API를 사용해 비동기적으로 파일을 읽어들여 퀴즈 로직에 매핑.

### ② 3가지 학습 모드 제공
*   **플래시카드 모드 (Flashcard)**: 영어 문장을 보여주고, 클릭 시 부드러운 3D 회전 애니메이션과 함께 한글 뜻 노출.
*   **문답 퀴즈 모드 (Typing Quiz)**: 한글 뜻을 보고 올바른 영어 문장을 직접 입력. 오타 체크 및 실시간 피드백 제공.
*   **리스닝 모드 (TTS Read)**: 브라우저 내장 Web Speech API(SpeechSynthesis)를 활용하여 원어민 발음으로 영어 문장을 읽어주는 흘려듣기 기능 지원 (유튜브 재생 효과 대체).

### ③ 개인화 진행 상황 및 학습 제어
*   학습 진행률 표시 (예: `45 / 1000 문장 완료`).
*   북마크(오답 노트) 기능: 잘 외워지지 않는 문장을 표시하고 별도로 모아서 학습 가능.
*   데이터는 `localStorage`에 자동 저장되어 브라우저를 새로고침해도 학습 이력이 유지됨.

---

## 4. UI/UX 디자인 시스템 (style.css)

*   **테마 컬러**: 심플한 다크 딥 블루(Dark Deep Blue, HSL 기반)를 기본 배경으로 하고, 세련된 아쿠아 블루와 바이올렛 파스텔 톤의 그라데이션을 포인트로 활용.
*   **효과**: 
    *   네온 글로우(Neon Glow) 효과를 이용한 모던한 카드 및 버튼 스타일.
    *   배경에 반투명한 블러 필터를 적용하는 글래스모피즘(Glassmorphism) 스타일 채택.
    *   3D Flip(뒤집기) 카드 트랜지션 애니메이션 적용.
*   **반응형 레이아웃**: 모바일 화면에서도 퀴즈를 쉽게 풀 수 있도록 반응형 패널 레이아웃 설계.

## 5. 단계별 구현 마일스톤

| 단계 | 작업 내용 | 목표 산출물 |
| :--- | :--- | :--- |
| **1단계** | 프로젝트 폴더 생성 및 `sentences.json` 샘플 DB 제작 | `sentences.json` (대표 30~50문장) |
| **2단계** | HTML 레이아웃 및 CSS 디자인 시스템(글래스모피즘) 스타일 완성 | `index.html`, `style.css` |
| **3단계** | JSON 비동기 fetch 로직 및 플래시카드/퀴즈 알고리즘 구현 | `app.js` |
| **4단계** | Web Speech API 연동(TTS) 및 로컬 스토리지 진도 저장 기능 탑재 | `app.js` 고도화 |
| **5단계** | 로컬 파이썬 서버 띄우기 실습 및 최종 교차 검증 | 앱 실행 및 테스팅 |
