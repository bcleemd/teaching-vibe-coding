# 초보자 대상 바이브코딩(Vibe Coding) 입문 교육 자료

본 교육 자료는 코딩을 한 번도 해보지 않았거나 기초 지식이 부족한 입문자가 AI 어시스턴트와 협업하여 직접 소프트웨어를 만들어 보는 **바이브코딩(Vibe Coding)** 입문 과정입니다. 

기초 개발 환경 세팅부터 브라우저 AI 활용, 에이전트 CLI(`agy`) 활용, 그리고 실제 로컬 실행 및 고도화 실습까지 아우르는 단계별 교안을 포함하고 있습니다.

---

## 📅 교육 커리큘럼

- **[1부: CLI 설치 및 기초 환경 설정 가이드](session1_cli_setup.md)**
  - 개발자의 기본 도구인 '터미널' 열기 & PowerShell 업그레이드
  - OS별 CLI 환경 세팅 (Node.js, Git, `uv` 패키지 매니저 및 Python 설치)
  - 필수 명령어와 디렉토리 이동 배우기
- **[2부: 바이브코딩 입문 및 첫 AI 협업 가이드](session2_vibe_coding_intro.md)**
  - 바이브코딩의 기본 개념 이해
  - AI를 내 개발 파트너로 길들이는 프롬프트 3대 법칙
  - 첫 협업 실습: 자기소개 웹페이지 작성 및 실행하기
- **[3부: AI 에이전트 CLI(agy) 설치 및 세팅 가이드](session3_cli_setting.md)**
  - AI 에이전트 CLI 도구의 필요성 및 가치 이해
  - 무료 사용자를 위한 `agy CLI` (Antigravity CLI) 검색 및 설치
  - 구글 계정 로그인 연동 및 첫 대화 테스트
  - (참고) Claude Code 및 Codex CLI 공식 설치 명령어 수록
- **[4부: AI 에이전트 실전 실습 - 영어 회화 1000문장 학습 앱](session4_vibe_coding_practice.md)**
  - 유튜브 1000문장 영어 흘려듣기를 모티브로 한 영어 Q&A 학습 앱 구동
  - 로컬 파일 fetch 시 발생하는 브라우저 CORS 문제 해결을 위한 파이썬 서버 띄우기 실습
  - 터미널 에이전트(`agy CLI`)에게 지시하여 소스 코드(JSON 데이터 추가, CSS 테마 수정, 기능 추가) 직접 제어하기 실습
- **[5부: CLI를 활용한 크로스 플랫폼(윈도우/맥) 자동화 작업 구축 가이드](session5_automation.md)**
  - 크로스 플랫폼 자동화 스크립팅 수단으로서의 파이썬 가치 학습
  - 파일 자동 복사 및 날짜별 저장 백업 파이썬 스크립트 작성 실습
  - macOS 스케줄러(`cron`) 및 Windows 스케줄러(`schtasks`)를 활용한 터미널 자동 예약 작업 등록
- **[6부: 터미널을 넘어 스마트폰으로 - 텔레그램 챗봇 연동 영어 학습 비서](session6_telegram_bot.md)**
  - 텔레그램 챗봇 생성 및 BotFather를 통한 API 봇 토큰 발급 절차 학습
  - AI 에이전트에 입력할 정밀 챗봇 기능 요구사항 정의 프롬프트 작성법
  - `/test` (진도 재개), `/review` (오답 노트), `/status` 명령어 설계 및 백그라운드 구동 실습
