# 4부: AI 에이전트 실전 실습 - 영어 회화 1000문장 학습 앱 제작 및 에이전트 수정 실습

2부에서는 브라우저 AI(ChatGPT/Claude)가 짜준 코드를 직접 마우스로 복사하여 VS Code에 붙여넣는 **수동 바이브코딩**을 진행했습니다.  
이번 4부에서는 터미널에 설치한 **`agy CLI` (AI 에이전트)**가 내 컴퓨터의 폴더와 파일을 직접 생성하고 수정하도록 지시하는 **진정한 에이전트식 바이브코딩(Agentic Vibe Coding)**을 실습합니다.

실습 예제로는 유튜브의 인기 영어 학습 시리즈인 [생활영어 1000문장 그냥 틀어두십쇼 1,2,3편 몰아듣기](https://www.youtube.com/watch?v=_50HS70urTw) 영상을 모티브로 한 **프리미엄 1000문장 Q&A 영어 회화 학습기**를 사용합니다.

---

## 1. 프로젝트 개요 및 구조

우리는 `session4_english_app` 폴더 내에 다음과 같은 모던 웹 애플리케이션을 완성해 두었습니다.

*   `sentences.json`: 원어민 핵심 일상 표현 50개의 영어 문장, 한글 해석, 상황별 꿀팁을 담은 JSON 데이터베이스.
*   `index.html`: 플래시카드 모드, 문답 퀴즈 모드, 연속 흘려듣기 모드를 전환할 수 있는 마크업 구조.
*   `style.css`: 반투명 유리에 비치는 듯한 **글래스모피즘(Glassmorphism)** 효과와 모던한 네온 블루 다크 테마 스타일시트.
*   `app.js`: JSON 데이터를 불러오고(Fetch), 퀴즈 정답 검증, 북마크 저장, 웹 브라우저 내장 음성 합성 API(Web Speech API)를 통한 원어민 TTS(음성 읽기) 오디오 처리 등 핵심 로직 담당.

## 2. [필수] 로컬 웹 서버 실행하여 앱 구동하기 (CLI의 필요성)

완성된 `index.html` 파일을 단순히 더블클릭해서 실행하면, 브라우저의 보안 정책(CORS 정책) 때문에 `sentences.json` 데이터 파일을 정상적으로 읽어오지 못하고 에러가 발생합니다.  
따라서 우리는 1부에서 준비한 **파이썬(Python/uv) 환경**을 통해 내 컴퓨터에 아주 가벼운 로컬 웹 서버를 실행해 주어야 합니다.

1. 터미널을 열고 본 프로젝트 폴더 안의 앱 디렉토리로 이동합니다:
   ```bash
   cd session4_english_app
   ```
2. 파이썬을 이용해 로컬 서버를 8000번 포트로 구동합니다:
   - **`uv` 패키지 매니저가 있을 때 (권장)**:
     ```bash
     uv run python -m http.server 8000
     ```
   - **기본 파이썬 설치 환경**:
     ```bash
     python -m http.server 8000
     ```
3. 크롬이나 사파리 등 인터넷 브라우저를 열고 아래 주소로 접속합니다:
   - **[http://localhost:8000](http://localhost:8000)**

> 🎉 **축하합니다!**  
> 모던하고 아름다운 UI의 영어 회화 학습기가 작동합니다. 좌측 메뉴에서 **플래시카드**, **문답 퀴즈**, **연속 흘려듣기** 모드를 바꾸어가며 원어민 음성과 함께 영어 학습을 즐겨보세요!

## 3. 에이전트(`agy CLI`)를 활용한 실시간 코드 수정 실습 (Vibe Coding)

이 웹 앱의 진정한 가치는 **내가 코드를 한 줄도 만지지 않고 터미널에 명령만 내려서 기능을 확장하고 바꿀 수 있다는 점**에 있습니다.  
터미널 창을 새로 하나 더 열어서(`session4_english_app` 폴더 위치) 아래 실습을 따라 해보세요.

### 🎯 실습 A: JSON 데이터베이스에 새로운 표현 추가 요청하기
현재 50개의 문장이 저장되어 있습니다. 에이전트에게 5개의 유용한 생활 표현을 추가해 달라고 지시해 봅니다.
*   **터미널 입력**:
    ```bash
    agy "sentences.json 파일에 전형적인 일상 생활 영어 표현 5개를 기존 포맷에 맞게 추가해줘."
    ```
*   **동작 관찰**: `agy`가 직접 `sentences.json`을 읽고, 파일의 대괄호`[]` 구조를 분석해 새로운 데이터 오브젝트를 추가하고 저장하는 과정을 터미널에서 지켜보세요.
*   **결과 확인**: 브라우저 화면을 새로고침하면 전체 문장 개수가 **55개**로 늘어난 것을 볼 수 있습니다!

### 🎯 실습 B: 다크모드 메인 테마 색상 변경 요청하기
현재 테마의 기본 포인트 컬러는 시안(Cyan) 블루입니다. 이를 싱그러운 에메랄드 그린 톤으로 변경해 보겠습니다.
*   **터미널 입력**:
    ```bash
    agy "style.css 파일의 포인트 컬러인 --accent-cyan과 --accent-cyan-glow 변수의 HSL 색상 값을 에메랄드 그린 색상 계열로 수정해줘."
    ```
*   **결과 확인**: 저장이 완료된 후 브라우저를 새로고침하면, 별도의 작업 없이 화면의 포인트 컬러와 버튼들의 네온 효과가 에메랄드 그린 색상으로 마법처럼 바뀌어 있는 것을 볼 수 있습니다.

### 🎯 실습 C: 새로운 기능 추가해보기
*   **터미널 입력**:
    ```bash
    agy "index.html과 app.js를 수정해서 퀴즈 모드에서 한글 뜻 옆에 영문 문장의 글자 수(예: 5 단어) 힌트를 추가로 보여주는 기능을 구현해줘."
    ```
*   **결과 확인**: 브라우저 새로고침 후 퀴즈 모드로 넘어가 한글 해석 옆에 단어 개수 힌트가 출력되는지 확인합니다.

## 4. 정리 및 바이브코딩 철학

직접 코드를 타이핑하며 오타와 씨름하는 대신:
1. **문제 정의 및 아이디어 기획** (개발자)
2. **에이전트에게 구현 지시** (개발자 $\rightarrow$ `agy`)
3. **코드 일괄 작성 및 디버깅** (`agy`)
4. **브라우저나 로컬 서버에서 최종 결과 확인** (개발자)

이것이 현대적인 **AI 에이전트 기반 바이브코딩**의 핵심 워크플로우입니다. 이 앱을 교두보 삼아 더 다채롭고 큰 기능들을 에이전트와 티키타카하며 만들어보세요!
